# Article files

Official title:

**Non-homologous isofunctional enzyme candidates in neurologically relevant helminths**

Expected manuscript formats:

- `Parasites_and_Vectors_format/`
- `Preprint_format/`
- `figures/`
- `supplementary/`
- `references/`

The complete DOCX/PDF manuscripts are included in the final local release package. The web repository currently contains the public text, data tables, workflow figure, CSV files, FASTA examples, Streamlit app and scripts. Binary DOCX/PDF/XLSX files should be uploaded through Git LFS, GitHub web upload, or standard git push from a local clone.

Required sections for the final manuscript:

- Title
- Abstract
- Keywords
- Background
- Methods
- Results
- Discussion
- Conclusions
- Availability of data and materials
- Competing interests
- Funding
- Authors' contributions
- Acknowledgements
- References
- Figure legends
- Supplementary information

Please cite the associated article if you use the data.
